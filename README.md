>11. Investigar ¿Qué es el pip y porque lo actualizamos?
pip es una herramienta de comandos que se utiliza para instalar y gestionar paquetes de software escritos en Phyton. 
Actualizamos el pip para:
    Seguridad # Las actualizaciones a menudo incluyen parches de seguridad.
    Nuevas caracteristicas # Cada version de pip, puede incluir nuevas características y mejoras que facilitan la gestión de paquetes.
    Compatibilidad # Las versiones más recientes de pip garantizan una mejor compatibilidad con las últimas versiones de Python y los paquetes de PyPI.
    Corrección de errores # Las actualizaciones a menudo corrigen errores y problemas presentes en versiones anteriores, mejorando la estabilidad y el rendimiento.
    Mejora del rendimiento # Las actualizaciones pueden incluir optimizaciones que hacen que pip sea más rápido y eficiente.
